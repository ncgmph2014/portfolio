Technical Overview

I added a backend feature to legacy code inherited from another software engineer who developed a React.js app. I added a button to the app which stored data about the dresses on the backend.

Technical: Backend

I utilized Node.js, Express.js, and MySQL in order to store data about the user's favorite dresses.

Technical: Frontend

I created a favorite button using React.js that would then communicate witht the backend to store data about the user's favorite dress(es).
